# Background animation snapshot

This directory contains the complete state immediately before background/fluid
interaction was added.

- Open `index.html` in this directory to view the saved version.
- In the current version, append `?interaction=off` to the URL to disable the
  interaction without restoring files.
- To restore permanently, copy `index.html`, `style.css`, `fluid.js`, and
  `hero.js` from this directory back to `background-animation/`.
